90EUDK - 90Edges Universe Development Kit

This is partial public distribution of our NEU Engine (NinetyEdgesUniverse Engine), an experimental project which we used in Ludum Dare 27 Jam. Feel free to play with it but do not expect too much from this version (0.0.2-beta). The only current "documentation" is the example inside .zip file (start with main.lua). Later we will use this Development Kit in our space game (check out 90Edges Universe website or see video below). Future versions of NEUDK will also allow you to quickly build similar space games.

Prerequisite

Windows: Current version is dependent on OpenAL. Now you have to "google" for it because Creative's link (http://connect.creativelabs.com/openal/Downloads/oalinst.zip) seems to be broken at this moment! We will distribute oalinst.exe inside .zip file in the next version (90EUDK 0.0.2-beta).
Mac OS X: none

Features

  • Simple sprite engine (for 2D games)
  • Basic sound support (.wav files)
  • Draw text support (Photoniacs Procedural Font - check it out on YouTube)
  • Keyboard input support
