90EUDK - 90Edges Universe Development Kit

This is our experimental project which we will use in Ludum Dare 27 Jam. Feel free to play with it but do not expect too much from this initial version (0.0.1-beta). The only current documentation is the example inside .zip file (start with main.lua).
