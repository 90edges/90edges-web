90EUDK - Documentation

90EUDK 0.0.1-beta

Script functions

-

mode1_createtexture( filename )

filename: name of texture file

return value: texure ID of new texture or 0

-

int mode1_getkeystate( lKeyCode )

lKeyCode: 0-up, 1-down, 2-left, 3-right, 32-space

return value: the return value specifies whether the key is pressed (0-no, 1-yes)

-

float mode1_getsoundvolume( channel )

channel: number of requested channel (from 1 to 255)

retun value: value from 0.0 to 1.0 (1.0 is max volume)

-

mode1_releasesound( channel )

channel: number of requested channel (from 1 to 255)

retun value: none

-

mode1_setsoundvolume( channel, volume )

channel: number of requested channel (from 1 to 255)
volume: new sound volume value from 0.0 to 1.0 (0.5 is 50%)

retun value: none
